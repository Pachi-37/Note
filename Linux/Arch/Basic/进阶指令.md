# 进阶指令

### df指令

- 查看磁盘空间
- df -h         以可读性较高形式

### free

- 查看内存使用

- free  -m         -m表示以mb形式

- ###### swap——临时内存转换

### head

- 查看一个文件前n行，不指定n默认为10行
- head  -n   文件路径

### tail

- 查看文件末尾行数——类似于head
  - tail   -n    文件路径
- tail    -f   查看一个文件动态变化
  - tail    -f     文件路径

### less

- 表示以较少内容输出，按下辅助功能键
  - 数字  空格   上下
- less     文件路径

### more

- 用法与less相似
- 但是不支持后退
